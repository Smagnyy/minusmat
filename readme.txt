Требования к запуску:

версия python - 3.11, устанавливать через Microsoft Store (для возможности запуска скриптов через cmd)

скачать с сайта https://alphacephei.com/vosk/models языковую модель (например vosk-model-ru-0.42)
языковая модель должна находиться в папке, в котором находится скрипт

скачать ffmpeg (https://www.gyan.dev/ffmpeg/builds/) 
желательно переименовать его папку в FFmpeg и указать Путь-до-папки\FFmpeg\bin в переменные среды->переменные среды ПОЛЬЗОВАТЕЛЯ->Path 

Библиотеки:
1. pip install ffmpeg
2. pip install ffmpeg-python
3. pip install vosk
4. pip install pydub

команды нужно просто прописывать в cmd
при установке библиотеки может писаться желтым цветом путь, по которому установилась библиотека, и что она не является PATH 
нужно как и в случае с FFmpeg скопировать этот путь и прописать его в переменных среды
